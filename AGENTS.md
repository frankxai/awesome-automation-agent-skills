# Agent Instructions

This repo is a public curated index. Keep it generic, useful, and safe.

## Rules

- No live workflow exports.
- No secrets, webhook URLs, credential names, private local paths, or customer data.
- Prefer links, short summaries, and mock examples.
- Every listed item should state the tool, use case, risk gate, and validation step.
- Run a public-safety scan before publishing whenever a scan script is available.
